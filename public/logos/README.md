# Logos de franquicias

Coloca aquí los logos de las franquicias como archivos `.svg`, `.png` o `.jpg`.

- Nombres sugeridos: `/<token>.svg` donde `<token>` es la clave normalizada (por ejemplo, `kfc`, `burger-king`, `mc-donald-s`, `casta`).
- Alternativamente, mapea nombres exactos en `src/logoMap.js` si el archivo tiene un nombre distinto o si usas una URL remota.

Importante: Asegúrate de tener derechos para usar estos logos.
